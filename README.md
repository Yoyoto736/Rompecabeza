# Rompecabezas de Máscara Geométrica - Solucionador Exacto

Este proyecto utiliza un algoritmo de *Backtracking* optimizado para encontrar todas las combinaciones posibles de empaquetado de piezas de madera en un tablero de 8x8, dejando expuesto un único color objetivo.

## Estructura del Proyecto
- `src/`: Contiene el motor matemático y las configuraciones de las piezas.
- `app/`: Contiene la interfaz gráfica desarrollada en Streamlit.
